## 🚀 Getting Started (3 Steps)

### Step 1: Supabase Setup (8 mins)
```
1. Go to https://supabase.com
2. Create free account
3. Create new project
4. Copy Project URL and Anon Key
5. Follow SQL commands in SETUP.md
6. Done! ✅
```

### Step 2: Configure & Install (5 mins)
```bash
cp portfolio-project/.env.example portfolio-project/.env.local
# Edit .env.local with your Supabase credentials

cd portfolio-project
npm install
```

### Step 3: Run & Deploy (5 mins)
```bash
npm run dev
# Test at http://localhost:3000/admin/login

# When ready:
npm run build
# Deploy dist/ to Vercel or Netlify
```

**Total time: 18 minutes to see your site running locally**
